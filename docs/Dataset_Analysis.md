# Dataset Analysis

## Dataset Name
Disease Prediction Using Machine Learning

## Files Used
- Training.csv
- Testing.csv

## Purpose
This dataset is used to train a machine learning model that predicts diseases based on symptoms entered by the user.

## Dataset Structure
- Input: Symptom columns (0 = symptom absent, 1 = symptom present)
- Output: prognosis (the disease to be predicted)

## Verification
- Dataset opened successfully.
- The last column is `prognosis`.
- Symptom values are 0 and 1.
- Dataset is ready for machine learning.
