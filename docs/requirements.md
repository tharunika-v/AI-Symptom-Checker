# Project Requirements

## Functional Requirements

1. User should be able to enter symptoms.
2. System should process the entered symptoms.
3. Machine learning model should predict the possible disease.
4. System should display the prediction result.
5. User should be able to interact with the application through a simple interface.

## Non-Functional Requirements

1. The system should provide quick responses.
2. The application should be easy to use.
3. The prediction system should be reliable.
4. The project should maintain proper data organization.
5. The system should be scalable for future improvements.

## Software Requirements

- Python
- Machine Learning Libraries
- Frontend Technologies
- Dataset Files

## Hardware Requirements

- Computer/Laptop
- Minimum 4GB RAM
- Internet connection for development and updates
