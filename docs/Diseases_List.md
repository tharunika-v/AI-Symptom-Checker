# Disease List

## Dataset Name
Disease Prediction Using Machine Learning

## Purpose
This file contains the list of diseases available in the dataset. These diseases are the possible outputs predicted by the machine learning model.

## Disease Classes

The dataset contains multiple disease categories. Each disease is represented in the `prognosis` column.

Examples of diseases:
- Fungal infection
- Allergy
- GERD
- Chronic cholestasis
- Drug reaction
- Peptic ulcer disease
- Diabetes
- Hypertension
- Migraine
- Pneumonia
- Common cold
- Hepatitis
- Malaria
- Dengue
- Tuberculosis

## Usage
The machine learning model uses symptom inputs to classify and predict one of these diseases.
