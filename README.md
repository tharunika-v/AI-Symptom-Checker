# AI Symptom Checker

## Overview
The AI Symptom Checker is a machine learning-based web application that predicts possible diseases based on symptoms entered by the user. It is designed to provide quick preliminary health insights using a trained machine learning model.

## Features
- Symptom-based disease prediction
- User-friendly interface
- Machine learning-powered predictions
- Dataset-driven analysis

## Technologies Used
- Python
- Machine Learning
- HTML
- CSS
- JavaScript
- Git & GitHub

## Project Structure
- `dataset/` – Training and testing datasets
- `docs/` – Project documentation
- `frontend/` – User interface files
